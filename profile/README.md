# 🧡 My Ability Share 🪐

미림마이스터고 전공 동아리 **MAS (My Ability Share)** 는 미림마이스터고등학교의 전공 동아리로,
다양한 **프로젝트**와 **공모전**에 참여하며 전공 역량을 기르고,
**소통과 협력**을 통해 함께 성장하고 끊임없이 도전하는 것을 목표로 합니다.

---

## 👀 MAS는 어떤 동아리인가요?

* 💻 전공 프로젝트 기획 & 개발
* 🏆 교내외 공모전 / 해커톤 참가
* 🧠 코드 리뷰 & 기술 세미나
* 🎯 진로 고민 공유 & 포트폴리오 피드백

전공을 **깊이 배우고**, 팀원과 함께 **성장하는 경험**을 쌓을 수 있습니다!
